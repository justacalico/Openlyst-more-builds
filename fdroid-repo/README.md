# OpenLyst F-Droid Repository

This is an F-Droid repository for Android applications from [OpenLyst](https://openlyst.ink).

## Installation

To add this repository to F-Droid:

1. **Open F-Droid** on your Android device
2. Go to **Settings** â†’ **Repositories**  
3. Tap the **+** icon to add a new repository
4. Enter this URL:

```
https://raw.githubusercontent.com/HttpAnimation/Openlyst-more-builds/main/fdroid-repo
```

5. Tap **Add** and wait for the repository to sync

## Available Applications

The repository contains free and open source Android applications from the OpenLyst project. All applications are available as APK downloads.

## Repository Structure

- `metadata/` - F-Droid metadata files for each application
- `repo/` - APK files and signatures (when available)
- `index.json` - Repository index for F-Droid client
- `icon.png` - Repository icon

## Manual Installation

You can also download APK files directly from the OpenLyst website or GitHub releases:

- [OpenLyst Apps](https://openlyst.ink/apps)
- [GitHub Releases](https://github.com/HttpAnimation/Openlyst-more-builds/releases)

## Security

F-Droid repositories use cryptographic signatures to ensure the authenticity and integrity of applications. This repository provides metadata for applications available through OpenLyst.

## Contributing

## Automated Updates

The repository metadata is automatically updated via GitHub Actions when the "Build F-Droid Repository" workflow runs.
